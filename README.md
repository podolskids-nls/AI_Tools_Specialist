## Описание

Тестовое задание: загрузка заказов в RetailCRM и синхронизация с Supabase.

## Что сделано

- Загрузил 50 тестовых заказов из mock_orders.json в RetailCRM через API
- Настроил синхронизацию RetailCRM → Supabase (таблицы orders и order_items)
- Сделал дашборд с графиком заказов по городам, задеплоил на Render
- Настроил Telegram-бота с уведомлениями о заказах от 50 000 ₸

## Стек

Python, RetailCRM API, Supabase, Flask, Telegram Bot API, Render

## Решения

Webhook в RetailCRM недоступен на бесплатном тарифе — заменил на поллер,
который проверяет новые заказы каждые 60 секунд.

Для проверки добавил 51-й заказ на сумму 70 000 ₸ — уведомление в боте пришло.
